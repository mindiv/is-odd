# Utils 🛠️

[![npm version](https://img.shields.io/npm/v/@mindiv/utils)](https://www.npmjs.com/package/@mindiv/utils)

### Utilities

- `isOdd(n)`
- `isEven(n)`
- `isPrime(n)`
- `isDivisibleBy(n, divisor)`
- `isMultipleOf(n, base)`
- `mod(n, m)` — like `%` but always positive
- `clamp(n, min, max)`
